As a result of the Alliance for Open Media (AOMedia), the AVIF still picture format was developed using the AV1 video format keyframes. Netflix has previously stated that the AVIF image format provides better image quality than JPEG, PNG, and even the newer WebP formats to reduce file size. It is time for a new, more advanced alternative to JPEG, making AVIF an exciting development with great adoption potential.
